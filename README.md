# Web-Based Internship Information System

## 🎓 Final Capstone Project

This is my **final capstone project** and the first full website I have created.  
It is a **Web-Based Internship Information System** designed to manage internship applications, postings, and student-company interactions in a centralized platform.

---

## 🚀 Project Overview

The system allows students, companies, and administrators to efficiently manage internship-related processes online.

It was built using **plain PHP, MySQL, JavaScript, CSS, and Bootstrap**, without any modern frameworks.

This project also uses the **Ninestars - Free Bootstrap Theme for Creative Template** for its UI design.

---

## 🛠️ Technologies Used

- PHP (Core / Plain PHP)
- MySQL (Database)
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Ninestars Bootstrap Theme

---

## 👥 User Roles

### 👨‍🎓 Student

- View available internship postings
- Apply for internships
- Track application status

### 🏢 Company

- Post internship opportunities
- Review student applications
- Accept or reject applicants

### 🧑‍💼 Administrator

- Manage users (students & companies)
- Approve internship postings
- Monitor system activity

---

## ✨ Features

- User authentication (Login/Register)
- Role-based access system
- Internship posting management
- Internship application system
- Application status tracking
- Admin dashboard
- Responsive UI using Bootstrap
